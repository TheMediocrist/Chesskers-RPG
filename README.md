# Welcome to the great game of Chesskers
But what *is* chesskers, you might ask? Chesskers is the most strategic and beautiful board game ever created.  So what is Chesskers RPG? Chesskers RPG builds on the infinite strategy and beauty of Chesskers, while adding complexity and hyper-realistic simulations of commanding troops in battle.

Another question that many may ask is "Are you sure Chesskers RPG is strategic, complex, or hyper-realistic when the code does ***nothing?***"  Well... it's new, okay?
